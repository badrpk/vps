# VPS-PK Cloud Platform ☁️

Open-source cloud services platform (API + website).

## Monorepo

| Path | What |
|------|------|
| `apps/api` | Node/Express cloud API (`vps-pk`) |
| `apps/www` | Marketing / docs site (`vps-pk.com`) |
| `docs` | Business & production notes |

## Download & use

```bash
git clone https://github.com/badrpk/vps.git
cd vps-pk/apps/api
npm install
npm start   # or: node server.js
```
